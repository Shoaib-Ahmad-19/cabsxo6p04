# Basic Security Policy for Small Office Setup :

A) Purpose :

This policy aims to protect the office’s information systems, data, and network resources from unauthorized access, misuse, and cyber threats.

B) Scope :
    This policy applies to:
        => All employees,Contractors and temporary staff.
        => All office computers, laptops, and network devices

1. Password Policy
    1.1 Password Requirements
        => Minimum 8–12 characters
        => Must include uppercase, lowercase, numbers, and special characters
        => Avoid common words and personal information

    1.2 Password Management
        => Passwords must be changed every 60–90 days
        => Do not reuse old passwords
        => Sharing passwords is strictly prohibited
        => Enable Multi-Factor Authentication (MFA) where possible

    1.3 Account Security
        => Lock account after 3–5 failed login attempts
        => Disable accounts of employees who leave the organization

2. Data Backup Policy
    2.1 Backup Frequency
        => Important data must be backed up daily
        => Full system backup at least once per week

    2.2 Backup Storage
        Maintain backups on:
            => External hard drives (offline backup)
            => Secure cloud storage
            => Keep at least one backup offsite

    2.3 Backup Security
        => Backup data must be encrypted
        => Access to backup files restricted to authorized staff
        => Regularly test backups to ensure data can be restored

3. Internet Usage Guidelines
    3.1 Acceptable Use
        => Internet should be used primarily for business purposes
        => Limited personal use allowed during breaks

    3.2 Prohibited Activities
        => Downloading unauthorized software
        => Visiting malicious or inappropriate websites
        => Sharing confidential company data online
        => Clicking suspicious email links or attachments

    3.3 Email Security
        => Do not open unknown attachments
        => Report phishing emails to IT/admin
        => Use official email accounts for work communication

4. Compliance & Monitoring
        => Office reserves the right to monitor network activity
        => Violations may result in disciplinary action
